Page Visibility API

Dec 6, 20191 min read view
#web api

document.hidden is a boolean value indicating if the current page is hidden

document.visibilityState will return a string indicating the current state, one of visible, hidden, and prerendered

visibilitychange event will fire when any of these changes, e.g. when the user opens you app's tab, or moves away from it.


Simply copy following code and run in dev tools console:

document.addEventListener('visibilitychange', (e) => {
  console.log(e.target.visibilityState)
})
